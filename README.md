# Health Sync Support

Public support, privacy, and security documents for Health Sync.

Health Sync exports the Apple Health records you authorize into machine-readable files in your own iCloud Drive.

## Product Boundary

- Manual export, HealthKit access, category selection, and iCloud file writing remain available without Pro.
- Optional Pro may enable automatic background export cadence.
- The developer does not operate a server that receives your export files.
